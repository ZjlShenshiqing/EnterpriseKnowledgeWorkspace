# Test Plan 测试计划

## 1. 测试范围

测试范围包括：

1. 登录与认证。
2. 用户和权限管理。
3. 知识库。
4. 智能问答。
5. 会议预约。
6. 日历与时间管理。
7. 待办事项。
8. 任务协同。
9. 消息通知。
10. 数据看板。
11. 系统管理。

## 2. 测试类型

1. 功能测试。
2. 权限测试。
3. 接口测试。
4. 集成测试。
5. 安全测试。
6. 性能测试。
7. 智能问答安全测试。
8. 回归测试。
9. 用户验收测试。

## 4. 智能问答安全测试

1. 答案不能包含无权限文档内容。
2. 答案不能暴露无权限文档标题。
3. 答案应包含来源引用。
4. 答案不应编造制度细节。
5. 系统应能防范文档中的提示词注入。
6. 证据不足时应返回无答案提示。

## 5. 性能目标

1. 首页加载时间控制在 3 秒以内。
2. 知识搜索响应时间控制在 2 秒以内。
3. 会议冲突检测控制在 2 秒以内。
4. 智能问答首响尽量控制在 5 秒以内。
5. MVP 阶段支持至少 500 名内部用户并发使用。

## 6. 验收标准

系统满足以下条件可以进入验收：

1. 所有 P0 功能通过功能测试。
2. 权限测试通过。
3. 智能问答来源和权限测试通过。
4. 会议冲突检测正常。
5. 消息通知正常。
6. 核心看板数据正确。
7. 不存在阻塞级或严重级缺陷。
