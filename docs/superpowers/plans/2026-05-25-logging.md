# 全模块日志系统实施计划

> **For agentic workers:** 使用 subagent-driven-development（推荐）或 executing-plans 按任务逐步实施。步骤使用 checkbox（`- [ ]`）语法跟踪。

**目标：** 为全部 4 个微服务 + 网关 + frameworks 建立统一日志体系——logback 文件输出 + AOP 请求日志 + 关键业务日志。

**架构：** 在 frameworks-web-starter 中新增 AOP 切面自动记录 Controller 请求日志，每个模块新增 logback-spring.xml 配置文件，所有 Controller 和 Service 类补加 `@Slf4j` 和关键操作日志。

**技术栈：** SLF4J + Logback + Spring AOP (aspectjweaver) + Lombok @Slf4j

- [ ] **Step 1: 创建 gateway logback-spring.xml**

```xml
<?xml version="1.0" encoding="UTF-8"?>
<configuration>
    <!-- 日志目录 -->
    <springProperty name="LOG_DIR" source="logging.file.path" defaultValue="logs"/>
    <springProperty name="APP_NAME" source="spring.application.name" defaultValue="gateway"/>

    <!-- 控制台输出（带颜色） -->
    <appender name="CONSOLE" class="ch.qos.logback.core.ConsoleAppender">
        <encoder>
            <pattern>[%d{yyyy-MM-dd HH:mm:ss.SSS}] [%highlight(%-5level)] [%X{traceId}] [%logger{36}:%line] %msg%n</pattern>
        </encoder>
    </appender>

    <!-- 全量日志文件，按天滚动 -->
    <appender name="FILE" class="ch.qos.logback.core.rolling.RollingFileAppender">
        <file>${LOG_DIR}/${APP_NAME}.log</file>
        <encoder>
            <pattern>[%d{yyyy-MM-dd HH:mm:ss.SSS}] [%-5level] [%X{traceId}] [%logger{36}:%line] %msg%n</pattern>
        </encoder>
        <rollingPolicy class="ch.qos.logback.core.rolling.SizeAndTimeBasedRollingPolicy">
            <fileNamePattern>${LOG_DIR}/%d{yyyy-MM-dd}/${APP_NAME}-%d{yyyy-MM-dd}-%i.log.gz</fileNamePattern>
            <maxFileSize>50MB</maxFileSize>
            <maxHistory>30</maxHistory>
        </rollingPolicy>
    </appender>

    <!-- ERROR 单独输出 -->
    <appender name="ERROR_FILE" class="ch.qos.logback.core.rolling.RollingFileAppender">
        <file>${LOG_DIR}/${APP_NAME}-error.log</file>
        <filter class="ch.qos.logback.classic.filter.ThresholdFilter">
            <level>ERROR</level>
        </filter>
        <encoder>
            <pattern>[%d{yyyy-MM-dd HH:mm:ss.SSS}] [%-5level] [%X{traceId}] [%logger{36}:%line] %msg%n</pattern>
        </encoder>
        <rollingPolicy class="ch.qos.logback.core.rolling.SizeAndTimeBasedRollingPolicy">
            <fileNamePattern>${LOG_DIR}/%d{yyyy-MM-dd}/${APP_NAME}-error-%d{yyyy-MM-dd}-%i.log.gz</fileNamePattern>
            <maxFileSize>50MB</maxFileSize>
            <maxHistory>30</maxHistory>
        </rollingPolicy>
    </appender>

    <root level="INFO">
        <appender-ref ref="CONSOLE"/>
        <appender-ref ref="FILE"/>
        <appender-ref ref="ERROR_FILE"/>
    </root>

    <!-- 框架日志降噪 -->
    <logger name="com.alibaba.nacos" level="WARN"/>
    <logger name="org.springframework.cloud.gateway" level="WARN"/>
</configuration>
```

#### Task 3: 创建 ControllerLogAspect

**文件：**

- [ ] **Step 1: 创建切面类**

```java
package com.zjl.common.logging;

import com.fasterxml.jackson.core.JsonProcessingException;
import com.fasterxml.jackson.databind.ObjectMapper;
import jakarta.servlet.http.HttpServletRequest;
import lombok.extern.slf4j.Slf4j;
import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.Around;
import org.aspectj.lang.annotation.Aspect;
import org.aspectj.lang.annotation.Pointcut;
import org.aspectj.lang.reflect.MethodSignature;
import org.springframework.boot.autoconfigure.condition.ConditionalOnProperty;
import org.springframework.stereotype.Component;
import org.springframework.web.context.request.RequestContextHolder;
import org.springframework.web.context.request.ServletRequestAttributes;

import java.util.HashMap;
import java.util.Map;
import java.util.Set;

@Slf4j
@Aspect
@Component
@ConditionalOnProperty(value = "app.logging.controller.enabled", havingValue = "true", matchIfMissing = true)
public class ControllerLogAspect {

    private static final ObjectMapper MAPPER = new ObjectMapper();
    private static final Set<String> SENSITIVE_KEYS = Set.of(
        "password", "passwd", "pwd",
        "token", "accessToken", "refreshToken",
        "secret", "secretKey", "authorization"
    );
    private static final int MAX_BODY_LENGTH = 500;

    @Pointcut("within(@org.springframework.web.bind.annotation.RestController *) || within(@org.springframework.stereotype.Controller *)")
    public void controllerPointcut() {}

    @Around("controllerPointcut() && execution(public * *(..))")
    public Object logController(ProceedingJoinPoint joinPoint) throws Throwable {
        long start = System.currentTimeMillis();
        MethodSignature signature = (MethodSignature) joinPoint.getSignature();
        String methodName = signature.getMethod().getName();
        String className = signature.getDeclaringType().getSimpleName();

        ServletRequestAttributes attrs = (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
        String httpMethod = "?";
        String uri = "/?";
        if (attrs != null) {
            HttpServletRequest request = attrs.getRequest();
            httpMethod = request.getMethod();
            uri = request.getRequestURI();
        }

        String args = maskSensitive(toJson(joinPoint.getArgs()));

        log.info("→ {} {} | {}.{} | args={}",
            httpMethod, uri, className, methodName,
            args.length() > MAX_BODY_LENGTH ? args.substring(0, MAX_BODY_LENGTH) + "..." : args);

        try {
            Object result = joinPoint.proceed();
            long elapsed = System.currentTimeMillis() - start;
            log.info("← {} {} 200 ({}ms)", httpMethod, uri, elapsed);
            return result;
        } catch (Exception e) {
            long elapsed = System.currentTimeMillis() - start;
            log.error("✗ {} {} 500 ({}ms): {}", httpMethod, uri, elapsed, e.getMessage(), e);
            throw e;
        }
    }

    private String toJson(Object obj) {
        try {
            return MAPPER.writeValueAsString(obj);
        } catch (JsonProcessingException e) {
            return String.valueOf(obj);
        }
    }

    @SuppressWarnings("unchecked")
    private String maskSensitive(String json) {
        String result = json;
        for (String key : SENSITIVE_KEYS) {
            result = result.replaceAll(
                "\"" + key + "\"\\s*:\\s*\"[^\"]*\"",
                "\"" + key + "\":\"***\"");
        }
        return result;
    }
}
```

**SimpleRateLimitGlobalFilter:**
1. 加 `@Slf4j`
2. 限流触发时加: `log.warn("限流触发: IP={}, path={}", ip, path)`

**IpAccessGlobalFilter:**
1. 加 `@Slf4j`
2. IP 黑名单拦截时加: `log.warn("IP 黑名单拦截: IP={}, path={}", ip, path)`

**IdentityPropagationGlobalFilter:**
1. 加 `@Slf4j`
2. 身份头注入时加: `log.debug("身份头注入: userId={}, deptId={}, downstream={}", userId, deptId, targetUri)`

**AuthController:**
1. 加 `@Slf4j`
2. 登录成功: `log.info("用户登录成功: userId={}, username={}", userId, username)`
3. 登录失败: `log.warn("用户登录失败: username={}, reason={}", username, reason)`

**SystemAdminController:**
1. 加 `@Slf4j`
2. 管理员操作（创建/删除用户等）: `log.info("管理员操作: {} {}, operatorId={}, targetId={}", httpMethod, uri, operatorId, targetId)`

**GatewayExceptionHandler:**
1. 加 `@Slf4j`
2. 异常时: `log.error("网关路由异常: {} -> {}, error={}", requestUri, targetService, e.getMessage(), e)`

**注意：** 网关是 WebFlux 非阻塞的，不能直接用 `HttpServletRequest` 获取请求信息。需要用 ServerWebExchange 对象获取。实际路径获取方式依具体类而定。

## 注意事项

- 网关是 WebFlux 响应式，不能用 `RequestContextHolder`——AOP 切面在网关下不会生效，需手动加网关日志
- 不要在日志中记录 JWT Token 原始值
- 入参 JSON 截断 500 字符，敏感字段自动 `***` 替换
- 文件日志每天按 50MB 切分，保留 30 天
